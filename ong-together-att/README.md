# ONG Together 

Versão Single-Page Application criada a partir do projeto de ONGs sustentáveis.

## Como usar

1. Coloque as pastas `css/` e `img/` na mesma pasta de `index.html`.
2. Abra `index.html` no navegador.
3. Navegue via links: `#home`, `#projetos`, `#cadastro`.
4. O formulário de cadastro salva no `localStorage` (apenas demo local).

## Observações

- Para backend (salvar cadastros em servidor) é preciso criar uma API.
