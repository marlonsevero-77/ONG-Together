# ONG Together 

Versão criada a partir do projeto de ONGs sustentáveis.

## Como usar

1. Coloque as pastas `css/` e `img/` na mesma pasta de `index.html`.
2. Abra `index.html` no navegador.
3. Navegue via links: `index`, `projetos`, `cadastro`.

## Observações

- Para backend (salvar cadastros em servidor) é preciso criar uma API.
